All Ball

2022-2024

  • Product Design
  • Branding

Comprehensive platform with a mobile app for team management and a web app for league administration. I designed the product to allow coaches, players, parents, and anyone involved to effortlessly manage teams, schedule practices and games, and track attendance. The web app provides league operators with powerful tools to oversee teams, venues, referees, and complex schedules, ensuring smooth operations across the board.

Team-Based Colors

The app's UI adapts to the team's colors. This feature helps coaches and managers quickly identify which team they’re working with, especially when managing multiple teams.

Practice Builder

Coaches often use personalized practice plans that focus on different aspects of the game. This required a practice builder that enables them to create custom practices by selecting drills from the library, or creating their own. Coaches can then integrate these custom plans into upcoming team practice events.

Library

We used distinct color schemes to differentiate between public, team, and personal content within the content library, making it easier for users to navigate and understand the context they’re in. This design choice ensures clarity as users move between public, team, and personal resources.

Live Game Stat Tracking

The stat tracker allows multiple users to record game events in real-time, with all data synced post-game and tied to the game’s summary. The main UI challenge was designing an intuitive interface that allowed for quick, accurate input during fast-paced gameplay.

Event Scheduler

The event scheduler handles the complexity of organizing games across various events, divisions, venues, and courts. It features a dynamic timeline with responsive game pills that adjust to different zoom levels.

One of the main challenges was designing a UI that could flexibly handle varying numbers of venues and courts, especially when hundreds of games might be visible simultaneously. It was crucial for the game pills to clearly convey any action needed, such as publishing or assigning staff, and to flag any issues like broken event rules, all at a glance.

This solution streamlined the scheduling process, making it efficient even for the most intricate league setups.
